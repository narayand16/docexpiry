# Tech stack

- React, NextJS, TypeScript, Tailwind CSS
- Supabase(Free tier) for backend implementation

## Important things to keep in mind

- Write readable code as a well-written prose. Make sure its maintainable
- Avoid duplicating the code. Keep functions modular
- Avoid exposing sensitive information via source code. Eg. Supabase secret key, MSG91 auth key, etc

### Whatsapp based OTP verification and reminders vis MSG91 provider is a must for this project
